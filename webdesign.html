Web Design

In 8th grade I took a web design 1 class and learned the basics of web design and HTML coding as well as CSS and a tiny bit of Java script. This year I took a web design 2 class and learned the more advanced ways of coding and how to make a website look good.

The First Website Project

This project was designed as an easy quick project to teach the new comers in class the basics of HTML and CSS. The objective was to create a 2-3 page website that had responsive properties and had 5-6 hyperlinks.

The Second Website Project

This project was rolled out in order to teach the noobs to write HTML and CSS properly. This time we were given content to make a website. This content had to be worked into the final product and had to include at least 90% percent of the original content. You could make your website about whatever you wanted but it had to include the given content. This project was easy and fun and I feel like it really helped the web 1 kids understand how to work with a client when requested to make a website.
