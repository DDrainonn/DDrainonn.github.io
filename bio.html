Biology At Compass

The biology class at Compass is pretty cool. I love biology and all kinds of life studies and the biology class at Compass is very Fun.

The Introductory Project

This project was the very first project that I ever presented at Compass. The point of this project was to come up with a slide show or any other kind of presentation and present it up on stage in front of the whole class stating who you are and where you come from. This project was very fun and challenging as well as one of the scariest things that I had to do at the time.I had never presented on stage and could barely bring myself to do it when it was my turn. In the end the project got done and was fun.

The Mars Project

With this project we where tasked with surviving on mars. We were going to ride a rocke to mar and had to come up with a plan to make a habitat, make food, make enrgy, and find water. This project was very fun and was challengin to study the different types of sciences going into living on mars.
